Most generator problems in Lagos start with the wrong size. People buy what a seller recommends, not what their appliances actually draw.

Start with a load list. Write down everything you want to run at the same time: fridge, freezer, fans, lights, router, TV, and any power tools or equipment. Each has a running wattage, and some (like fridges and pumps) pull extra for a second when they start.

Add the running wattages, then add a margin for those startup spikes. A 2kVA unit is enough for lights, fans, a fridge, and a router. A small shop with tools needs more, often 5kVA or above.

Diesel or petrol depends on how long you run it. Diesel suits longer daily use; petrol suits lighter, occasional use. Tell us your load list and we will recommend a capacity instead of selling you the biggest box.

The point is simple: size to the load, not to the seller's stock.
